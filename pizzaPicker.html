Technical Details

Thanks for checking out this project from my portfolio! This page is only visible if coming from there, so as to not ruin the 'immersion' that would come with this being a commercial product, so I appreciate you taking the time to read this page!

After finding the idea for this project, a general pizza order taker, I started looking for ways to make it more interesting. My inital thoughts were to find public APIs for different pizza chains to allow for up-to-date prices. These APIs are not made public, so I decided to create a JSON file with prices and pizza data that I found on their websites. (All prices and data are from Fort Collins, CO in 2021.)

Now that I knew I would have data to display, I thought about other features I wanted to include. I drew pictures of what different screens would look like, how to display a user's saved profiles, the home screen, etc. I wrote out how I expected to implement the back end, how I would use localStorage to easily create customizable user experiences without a server.

I created the front-end first, leaving placeholders for where I envisioned user data and JSON data would be later displayed. Then once all the pages were created and could be navigated between, I worked on filling them in with the data and writing the functions that would run in the back end to compute a final order to give to the user.

This little project was started from scratch, and written with HTML, CSS, and the jquery library of JavaScript to more easily manipulate the DOM, as well as get some practice with it as this is the first time I have used jquery. Things I learned from this project are, jquery, css transitions, reading form data without php (through js), creating a single page application with multiple views, and working with JSON data without React. At the time of writing, future plans include hosting this site on Amazon S3 and creating a db of users instead of storing them on the users machine.
