Project-grow website

Project Grow is a local Community Gardening organization, who we partnered with to create a brand-new website that would meet all of their needs. The site is written in HTML, CSS, and PHP, which lets us use the WordPress customization options to allow for easily customizable layouts. Pages include templated layouts that can be added or removed to accomodate the seasonal services they offer, as well as the rotating classes and events.

Our team of 5 developers is creating this website in a semester as our project. Pages were either designated to a single person or were given to groups of two, which created a sense of teamwork and experience working on projects in small groups. A link will be made available when it is finished.

Sudoku

This program solves pre-generated boards on the fly and allows playing Sudoku at the command line. Functionality includes playing 4 different difficulties, giving hints, ability to undo moves, see which numbers remain in each row, as well as see statistics about your game play. Future goals include creating a server which would allow statistics to persist across games, as well as a website which could host the game to create a more customizable layout than is offered at the command line.

This was created as a gift for my girlfriend. It is written in python and was developed to be as user-friendly as possible. This included many error-handling measures to ensure that she could easily undo anything she did, and provide an easy to use interface even when she is unfamilar with using a terminal.
